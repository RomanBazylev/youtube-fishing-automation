## Промпты для LLM

Этот файл описывает формат запросов к LLM, который используется в воркфлоу `YouTube Fishing Shorts Generator` (файл `n8n_workflow_youtube_fishing_generator.json`).

### 1. Генерация сценария ролика (Script LLM)

**Цель**: получить сценарий для вертикального ролика 30–60 секунд (формат Shorts) про рыбалку.

**Входные данные из n8n**:

- `topic` — тема ролика (из Google Sheets);
- `angle` — хук/подход;
- `keywords` — ключевые слова.

**Рекомендуемый промпт (логика в ноде `Build Script Prompt`)**:

```text
Ты сценарист YouTube-Shorts про рыбалку.
Сгенерируй короткий сценарий (30–60 секунд) на русском языке.

Формат вывода — строго JSON, без пояснений и лишнего текста.
Структура JSON:
{
  "intro":     { "text": "...", "image_hint": "..." },
  "tips":      [ { "text": "...", "image_hint": "..." } ],
  "cta":       { "text": "...", "image_hint": "..." }
}

intro.text  — цепляющее вступление (хук, максимум 1–2 предложения);
tips        — 2–4 коротких совета/факта, каждый в 1–2 предложениях;
cta.text    — призыв к действию (подписаться, лайк, поделиться, комментарий).

image_hint  — короткое описание кадра: что показать на видео (например: "лодка на рассвете", "крупный план щуки", "приманка над водой").

Тема: {{topic}}
Подход/хук: {{angle}}
Ключевые слова: {{keywords}}
```

**Ожидаемый ответ** — **валидный JSON** в одном из форматов:

```json
{
  "intro": { "text": "Текст вступления...", "image_hint": "лодка на заре" },
  "tips": [
    { "text": "Первый совет...", "image_hint": "поклёвка крупного окуня" },
    { "text": "Второй совет...", "image_hint": "замена приманки" }
  ],
  "cta": { "text": "Подпишись, если ловишь щуку!", "image_hint": "рыбак с уловом" }
}
```

Этот JSON далее используется:

- для построения текста озвучки (объединяются `intro.text`, все `tips[].text` и `cta.text`);
- для построения структуры кадров (`frames`) по полю `image_hint`.

### 2. Генерация метаданных YouTube (Meta LLM)

**Цель**: получить SEO‑оптимизированные заголовок, описание и теги для ролика.

**Входные данные из n8n**:

- `topic` — тема ролика;
- `keywords` — ключевые слова.

**Рекомендуемый промпт (логика в ноде `Build Meta Prompt`)**:

```text
Ты помогаешь оформить ролик на YouTube (Shorts) про рыбалку.
Сгенерируй строго JSON без пояснений и лишнего текста.

Структура JSON:
{
  "title":       "Короткий, цепляющий заголовок до 70 символов",
  "description": "Описание ролика в 1–2 абзацах, с ключевыми словами и хэштегами",
  "tags":        ["строка1", "строка2", "..."]
}

Требования:
- язык: русский;
- избегай кликбейта, но делай заголовок привлекательным;
- в description добавь 3–7 релевантных хэштегов (#рыбалка, #щука, и т.п.);
- tags — массив из 5–15 ключевых слов и фраз.

Тема: {{topic}}
Ключевые слова: {{keywords}}
```

**Ожидаемый ответ**:

```json
{
  "title": "3 ошибки при ловле щуки с лодки",
  "description": "Короткий текст описания... #рыбалка #щука #спиннинг",
  "tags": [
    "рыбалка",
    "щука",
    "спиннинг с лодки",
    "ошибки новичков",
    "советы по рыбалке"
  ]
}
```

Этот JSON сохраняется в файл `runtime/video_meta.json` и далее используется воркфлоу загрузки на YouTube.
